# QSliderX

## Features

- Node-based graphical interface
- Supports custom node definitions

## Build Instructions

1. Clone the repository:
   ```bash
   git clone --recurse-submodules https://github.com/otto-link/GNodeGUI.git
   ```
2. Build using CMake:
   ```bash
   mkdir build && cd build
   cmake ..
   make
   ```
